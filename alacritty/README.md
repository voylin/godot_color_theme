# Godot Color Theme - Alacritty
To create your alacritty.toml config, you run the Python script `generate_alacritty_toml.py` from within this folder.

## Usage
1. Run the script from your terminal `python3 generate_alacritty.py`;
2. Follow the prompts. You can press **Enter** to use the defaults;
3. The script will generate a file named `alacritty.toml` in the current directory;
4. Put the toml file in the correct location; (on Linux that would be `~/.config/alacritty/alacritty.toml`)
